TP3 — HTTP/3 HTTP/3 · QUIC · UDP · Encrypted

Protocol stack: HTTP/3 → QUIC (TLS 1.3 built-in) → UDP → IP
Encryption: TLS 1.3 integrated into QUIC — mandatory, cannot be disabled
Wireshark filter: quic or udp.port == 8443

Server is running. Use the Python client to generate QUIC traffic:

python client.py

What to observe in Wireshark

Packet typeDescription
QUIC InitialFirst UDP packet — contains TLS ClientHello (no TCP SYN!)
QUIC HandshakeTLS negotiation integrated into QUIC — 1 RTT instead of 2
QUIC 1-RTTEncrypted application data (HTTP/3 frames)
UDP datagramsAll QUIC traffic carried over UDP — no SYN/FIN/RST packets

Key observation: no TCP SYN visible — QUIC uses UDP. The TLS ClientHello is embedded in the very first QUIC Initial packet (1 RTT handshake).
