TP2 — HTTP/2 HTTP/2 · TLS 1.3 · Encrypted

Protocol stack: HTTP/2 → TLS 1.3 → TCP → IP
Encryption: TLS 1.3 — content is encrypted, only handshake metadata visible
Wireshark filter: tls or tcp.port == 8443

Server is running. Use the Python client to generate traffic:

python client.py

What to observe in Wireshark

Packet typeDescription
SYN / SYN-ACK / ACKSingle TCP handshake for the entire session
TLS ClientHelloClient proposes TLS version + cipher suites + ALPN (h2)
TLS ServerHelloServer selects parameters + sends certificate
TLS Application DataHTTP/2 frames — fully encrypted, unreadable in Wireshark

Key observation: only ONE TCP SYN visible for all requests. HTTP/2 multiplexes all requests over a single connection — compare with TP1.
