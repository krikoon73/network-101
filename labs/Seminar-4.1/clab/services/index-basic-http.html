⚠️ Basic HTTP Server

Server: basic-http-server
IP: 10.100.3.10
Protocol: HTTP (Port 80)
Encryption: NONE ❌

🔓 SECURITY WARNING

This connection is NOT encrypted!

All traffic between your browser and this server is transmitted in PLAIN TEXT.

What can an attacker see?

  • ✅ All URLs you visit
  • ✅ All form data (usernames, passwords, credit cards)
  • ✅ All cookies and session tokens
  • ✅ All API requests and responses
  • ✅ Everything you type or click

Educational Purpose

This server demonstrates why HTTPS is essential for production web applications.

Use Wireshark or tcpdump to capture this traffic and see how easy it is to read unencrypted HTTP!

Command: tcpdump -i eth3 port 80 -A

Available Endpoints

  • / - This page
  • /health - Health check endpoint
  • /api - JSON API endpoint
  • /hostname - Server hostname
  • /3mq.html - 📚 Les Trois Mousquetaires (Large file for TCP analysis)
  • /3mq.txt - 📄 Raw text file (4247 lines, ~500KB)
