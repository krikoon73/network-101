🔍 Educational Purpose

This server demonstrates the difference between encrypted (HTTPS) and unencrypted (HTTP) web traffic:

  • HTTP (this server): Data transmitted in plain text - anyone can intercept and read it
  • HTTPS (web-server): Data encrypted with TLS/SSL - protected from eavesdropping

In production environments, always use HTTPS to protect user data and privacy!
