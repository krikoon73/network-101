TP1 — HTTP/1.1 HTTP/1.1 · TCP · No Encryption

Protocol stack: HTTP/1.1 → TCP → IP
Encryption: None — all traffic is readable in Wireshark
Wireshark filter: tcp.port == 8080

Server is running. Use the Python client to generate traffic:

python client.py

What to observe in Wireshark

Packet typeDescription
SYN / SYN-ACK / ACKTCP 3-way handshake — one per chunk request
GET /data HTTP/1.1Client requests the data file
Range: bytes=X-YClient asks for a specific byte range
HTTP/1.1 206 Partial ContentServer responds with that chunk
FIN / ACKTCP connection closed after each chunk

Key observation: count the TCP SYN packets — there is one per chunk. HTTP/1.1 opens a new TCP connection for each request.
